# 416. 分割等和子集
## 题目描述
给你一个**只包含正整数**的 **非空** 数组 `nums` 。请你判断是否可以将这个数组分割成两个子集，使得两个子集的元素和相等。

### 示例
- 示例 1：
  输入：nums = [1,5,11,5]  
  
  输出：true  

  解释：数组可以分割成 [1,5,5] 和 [11]，两个子集和均为 11。

- 示例 2：
  输入：nums = [1,2,3,5]  

  输出：false  

  解释：数组总和为 11（奇数），无法分割为两个和相等的子集。

## 解题思路
### 1. 问题转化（核心）
题目要求分割为两个和相等的子集，等价于：
- 数组总和 `sum` 必须是**偶数**（奇数无法平分）；
- 能否从数组中找到一个子集，其和为 `target = sum / 2`（经典 0-1 背包问题）。

### 2. 边界条件提前过滤
提前判断无效场景，减少后续计算：
- 若 `sum` 是奇数 → 直接返回 `false`；
- 若数组中最大元素 `max_num > target` → 该元素无法被任何子集包含（否则子集和超过 `target`），直接返回 `false`。

### 3. 动态规划（二维 → 一维优化）
#### 二维 DP 思路（基础）
- 定义 `dp[i][j]`：前 `i` 个元素中，是否存在子集和为 `j`（值为 `true`/`false`）；
- 初始化：`dp[i][0] = true`（空集和为 0，恒成立）；
- 状态转移：
  - 若 `j >= nums[i]`：`dp[i][j] = dp[i-1][j] | dp[i-1][j-nums[i]]`（选/不选当前元素）；
  - 若 `j < nums[i]`：`dp[i][j] = dp[i-1][j]`（无法选当前元素，继承上一行结果）。

#### 一维 DP 优化（空间降维）
由于每一行仅依赖上一行结果，可压缩为一维数组：
- 定义 `dp[j]`：是否存在子集和为 `j`；
- 遍历顺序：**倒序遍历 `j`**（避免重复选取同一元素，符合 0-1 背包特性）；
- 状态转移：`dp[j] = dp[j] | dp[j-nums[i]]`（不选/选当前元素）。

## 代码
```cpp
class Solution {
public:
    bool canPartition(vector<int>& nums) {
        // 1. 计算数组总和
        int sum = accumulate(nums.begin(), nums.end(), 0);
        // 总和为奇数，直接返回false（无法平分）
        if (sum % 2 != 0) {
            return false;
        }

        // 2. 定义目标值 + 提前判断最大元素
        int target = sum / 2;
        int max_num = *max_element(nums.begin(), nums.end());
        if (max_num > target) {
            return false;
        }

        // 3. 一维DP数组初始化（dp[j]表示是否存在和为j的子集）
        vector<bool> dp(target + 1, false);
        dp[0] = true; // 空集和为0，恒成立

        // 4. 0-1背包核心逻辑（倒序遍历避免重复选元素）
        for (int num : nums) {
            for (int j = target; j >= num; --j) {
                // 选当前元素：dp[j-num]；不选：dp[j]；只要一个为true则dp[j]为true
                dp[j] = dp[j] || dp[j - num];
            }
        }

        // 5. 返回是否存在和为target的子集
        return dp[target];
    }
};
