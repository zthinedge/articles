# 72. 编辑距离

**难度**：中等
**相关标签**：相关标签、相关企业

给你两个单词 `word1` 和 `word2`，请返回将 `word1` 转换成 `word2` 所使用的最少操作数。

你可以对一个单词进行如下三种操作：
- 插入一个字符
- 删除一个字符
- 替换一个字符

---

## 示例 1：
**输入**：`word1 = "horse", word2 = "ros"`  
**输出**：`3`  
**解释**：  `horse -> rorse (将 'h' 替换为 'r')`  
`rorse -> rose (删除 'r')`  
`rose -> ros (删除 'e')`

---

## 示例 2：
**输入**：`word1 = "intention", word2 = "execution"`  
**输出**：`5`  
**解释**：`intention -> inention (删除 't')`  
`inention -> enention (将 'i' 替换为 'e')`  
`enention -> exention (将 'n' 替换为 'x')`  
`exention -> exection (将 'n' 替换为 'c')`  
`exection -> execution (插入 'u')`
## 思路
`dp[i][j]`代表`word1`中前`i`个字符转化为`word2`中前`j`个字符的最小操作次数。如果只对`word1`进行操作，如果它第`i`个字符和`word2`第`j`个字符相等，那么就不需要增加新的操作次数，值为`dp[i-1][j-1]`。  
如果不相等，有三种情况：  
1. 删除`word1`的第`i`个字符，那么操作次数就是`dp[i-1][j]+1`    
2. 插入一个字符匹配`word2`的第`j`个字符,那么`word1`第`i+1`个字符和`word2`第`j`个字符相等，所以操作次数为`dp[i][j-1]+1`     
3. 把`word1`的第`i`个字符换成`word2`的第`j`个字符,操作次数为`dp[i-1][j-1]+1`
## 代码
```cpp
class Solution {
public:
    int minDistance(string word1, string word2) {
        int m=word1.size();
        int n=word2.size();
        vector<vector<int>>dp(m+1,vector<int>(n+1));
        for(int i=1;i<=m;i++)dp[i][0]=i;
        for(int i=1;i<=n;i++)dp[0][i]=i;
        for(int i=1;i<=m;i++){
            for(int j=1;j<=n;j++){
                if(word1[i-1]==word2[j-1]){
                    dp[i][j]=dp[i-1][j-1];
                }else{
                    dp[i][j]=min(dp[i-1][j],min(dp[i][j-1],dp[i-1][j-1]))+1;
                }
            }
        }
        return dp[m][n];
    }
};
