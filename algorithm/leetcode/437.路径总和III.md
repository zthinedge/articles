# 437.路径总和III
## 题目描述
给定一个二叉树的根节点 `root` ，和一个整数 `targetSum` ，求该二叉树里节点值之和等于 `targetSum `的 `路径 `的数目。
`路径 `不需要从根节点开始，也不需要在叶子节点结束，但是路径方向必须是向下的（只能从父节点到子节点）。
### 示例
- 示例 1：
  输入：root = [10,5,-3,3,2,null,11,3,-2,null,1], targetSum = 8  

  输出：3  

  解释：和等于 `8 `的路径有` 3 `条

- 示例 2：
  输入：root = [5,4,8,11,null,13,4,7,2,null,null,5,1], targetSum = 22  

  输出：3
  
## 解题思路
要求节点之和等于 `targetSum`的路径的数目，那么需要知道每一个节点向下延申满足条件的总和。可以使用前缀和+哈希表，哈希表`unordered_map<long long ,int>prefix`中的两个参数指的是从根节点出发合为前缀和的路径的数目。
对于两个节点`i,j`，要使它们之间的路径（i的子节点到j节点）和为`targetSum`，则`prefix[j]-prefix[i]==targetSum` ,等价于`prefix[i]=prefix[j]-targetSum`，因此需要一个参数来记录当前节点的前缀和，结果加上`prefix[cur-targetSum]`,再递归该节点的左右子树，最后返回结果

## 代码
``` cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
    unordered_map<long long ,int>prefix;
public:
    int dfs(TreeNode *root,long long cur,int targetSum){
        if(!root)return 0;
        int res=0;
        cur+=root->val;
        if(prefix.count(cur-targetSum)){
            res+=prefix[cur-targetSum];
        }
        prefix[cur]++;
        res+=dfs(root->left,cur,targetSum);
        res+=dfs(root->right,cur,targetSum);
        prefix[cur]--;
        return res;
    }
    int pathSum(TreeNode* root, int targetSum) {
        prefix[0]=1;
        return dfs(root,0,targetSum);
    }
};
